Controllers & Extensions

Register custom controllers, capabilities, and presenters to extend mate.js

Controller Basics — mx-controller

Controllers manage stateful behavior. Register via window.ClassName = ClassName or mate.registerController(). Methods are invoked with @event:@controller:method.

0
▸ View code 
<div mx-controller="Counter">
  <button mx-click="@event:@controller:decrement">−</button>
  <span data-counter-display>0</span>
  <button mx-click="@event:@controller:increment">+</button>
</div>

Controller Registry API — mate.registerController()

Programmatic registration instead of polluting window. Check the browser console to see the registry in action.

Tick #0
▸ View code 
mate.registerController('RegistryController', class {
  constructor(node) {
    this.node = node;
    this.count = 0;
    this.display = node.querySelector('[data-registry-display]');
  }
  tick() {
    this.count++;
    this.display.textContent = 'Tick #' + this.count;
  }
});

Custom Capability — mate.registerCapability()

Register reusable logic with methods. The presenter renders the return value automatically.

Click a button to see the capability result
▸ View code 
mate.registerCapability('Analytics', {
  track(node, event, parsedEvent) {
    return Promise.resolve({ tracked: 'Analytics.track', timestamp: Date.now() });
  },
  identify(node, event, parsedEvent) {
    return Promise.resolve({ identified: true, user: 'demo-user' });
  },
});

// Simple function capability
mate.registerCapability('Logger', (node, method, event) => {
  return Promise.resolve({ level: method, message: 'Captured' });
});

Custom Presenter — mate.registerPresenter()

Define how responses are rendered. Two custom presenters registered below: @uppercase and @json.

@uppercase
Click button
@json (pretty-print)
Click button
▸ View code 
mate.registerPresenter('@uppercase', async (node, response) => {
  const text = await response.text();
  node.innerHTML = text.toUpperCase();
});

mate.registerPresenter('@json', async (node, response) => {
  const text = await response.text();
  const parsed = JSON.parse(text);
  node.innerHTML = '<pre>' + JSON.stringify(parsed, null, 2) + '</pre>';
});
