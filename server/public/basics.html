Basics

The three foundational patterns of mate.js

@request:@inner — Click to Load

The hello world of mate.js. Click a button, fetch HTML, replace the element's content.

▸ View code 
<button
  mx-click="@request:@inner"
  mx-path="/api/hello">
  Click me
</button>

mx-load — Auto-Load on Mount

Content loads immediately when the element enters the DOM. No user interaction needed.

Loading...
▸ View code 
<div mx-load="@request:@inner"
     mx-path="/api/hello">
  <span class="animate-pulse">Loading...</span>
</div>

mx-submit — Form Submission

Submit a form via POST. mate.js serializes the fields and replaces the form with the server response.

▸ View code 
<form mx-submit="@request:@inner"
      mx-method="POST"
      mx-path="/api/submit">
  <input type="text" name="username" />
  <input type="email" name="email" />
  <button type="submit">Submit</button>
</form>
